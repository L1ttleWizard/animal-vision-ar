# Мир глазами животного — AR-прототип

Мобильный веб-прототип AR-приложения **«Мир глазами животного»**. Демонстрирует, как одни и те же сцены воспринимаются разными животными: тепловое зрение змеи, ультрафиолетовое — пчелы, ночное — совы, дихроматическое — собаки и т.д. Приложение собирает за пользователем коллекцию открытых видов, начисляет «очки исследования» (валюта) и XP, ведёт дневник наблюдений.

Стек: **Next.js 16 (App Router) + React 19 + Redux Toolkit + TypeScript + Tailwind CSS v4**. Дизайн рассчитан под **iPhone 15 (393 × 852)** и центрируется во «фрейме телефона» при просмотре с десктопа.

> Это статичный прототип, а не реальная AR-камера. Все «AR-сцены» собраны на SVG/CSS — это иллюстративные эквиваленты, которые позже заменяются на WebXR/Three.js или нативный ARKit/ARCore.

## Архитектура и состояние

Redux Toolkit, со слайсами:

| Слайс           | Состояние                                                | Назначение                                          |
| --------------- | -------------------------------------------------------- | --------------------------------------------------- |
| `profile`       | имя, ранг, уровень, XP, монеты, друзья, наблюдения       | прогрессия пользователя                             |
| `gallery`       | список разблокированных животных (`AnimalId[]`)          | состояние коллекции моделей                         |
| `onboarding`    | `completed: boolean`, `step: number`                     | определяет, показывать ли первый экран              |
| `settings`      | флаги уведомлений / AR / приватности, язык интерфейса    | отвечает за toggles в `/profile/settings`           |

Всё состояние сериализуется в `localStorage` (`animal-vision-ar:v1`) на каждое изменение и подгружается через `preloadedState` при первом mount-е, чтобы избежать гидратационных конфликтов. Реализация — `src/store/Providers.tsx` без дополнительной зависимости `redux-persist` (на актуальных версиях Redux Toolkit и React 19 проще обойтись своим хелпером).

Хуки: `useAppDispatch`, `useAppSelector`, `useAppStore` объявлены в `src/store/hooks.ts` и типизированы.

### Источники данных

Все справочные данные лежат в `src/data/`:

* `animals.ts` — список животных, цены, описания, факты, палитры для AR-сцен.
* `chats.ts` — карточки чатов и треды сообщений.
* `games.ts` — описания мини-игр и список квестов / записей дневника.
* `tests.ts` — вопросы для теста и фигуры для игры на скорость.

## Дизайн-система

Цвета — глубокие тёмно-зелёные + золотые акценты (см. `globals.css`):

```
--color-bg-deep:   #0F1A0C
--color-bg-mid:    #1F2A1A
--color-bg-soft:   #2C3520
--color-gold-100:  #FFF1B8
--color-gold-200:  #F4D67A
--color-gold-300:  #D4AF6F
--color-gold-400:  #B8924E
--color-gold-500:  #8C6C36
--color-cream:     #F5EFD9
--color-ink:       #E9E2C8
--color-muted:     #9C9879
```

Шрифт — **Manrope** (через `next/font`), подтягивается с латиницей и кириллицей.

Готовые классы-утилиты:

* `.card-gold` — карточка с золотой обводкой и тонким градиентом.
* `.pill-gold` — пилюля-кнопка с эффектом подсветки на hover.
* `.glow-gold` — золотая подсветка текста.
* `.tab-active` — выделение активной вкладки нижней навигации.
* `.toggle-track` / `.toggle-thumb` — кастомный переключатель.
* `.shimmer-gold`, `.animate-sparkle`, `.page-fade` — фоновые анимации.

Декоративные слои (`<ScreenBackground />`):

* радиальный градиент глубины,
* мозаика отпечатков лап / птичьих следов (SVG),
* мерцающие звёзды,
* виньетка для подчёркивания краёв.

«Фрейм iPhone 15» — `<PhoneFrame />`: на десктопе показывает корпус с динамическим островком и индикатором Home; на мобильном превращается в полноэкранный контейнер (`max-width` 393 px).

## Локальный запуск

```bash
# Установка зависимостей
npm install

# Dev-сервер (Turbopack)
npm run dev
# → http://localhost:3000

# Прод-сборка
npm run build && npm run start

# Линт
npm run lint
```

Требования: Node 22+ (rекомендовано LTS), npm.

Стартовая страница `/` редиректит на онбординг при первом входе и в галерею при последующих. Чтобы пройти онбординг повторно — вручную перейдите в `/onboarding` или зачистите `localStorage`.

## Что вошло в прототип

* Полная навигация: 5 вкладок таб-бара + второстепенные экраны.
* Прогрессия пользователя: XP, уровень, монеты, разблокировка моделей.
* Локализация контента — RU (с заготовкой переключения языка в настройках).
* Мини-игра «Охота змеи» (тепловизор) с таймером и наградой.
* Тест на 5 вопросов + игра на скорость с наградой.
* Перс. состояние в `localStorage` (после онбординга и других действий — переживает перезагрузку).
* Адаптация под смартфон 393 × 852 и фрейм iPhone на десктопе.

## Что ещё не покрыто (заведомо)

* Реальная WebXR / ARKit / ARCore интеграция — все сцены SVG-имитации.
* Загрузка 3D-моделей и их кеширование.
* Серверная авторизация, друзья, мультиплеер чатов — мок-данные.
* Распознавание животных на фото — заглушка экрана «Определить».
* Поиск, лидерборды, нотификации — на будущих итерациях.
